# StudySense AI

StudySense AI is an AI-powered Study Planner and Progress Tracker built using the MERN Stack.

## Features

- User Authentication
- AI Study Planner
- Subject Management
- Assignment Tracker
- Exam Countdown
- AI Notes Generator
- AI Quiz Generator
- Analytics Dashboard

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Tailwind CSS
- Google Gemini API
